>
> Requires [Deno](https://deno.land/)

## Install

```sh
deno install --allow-read --allow-run --allow-write --allow-net -f -q --unstable https://deno.land/x/denon@2.3.2/denon.ts
```

## Usage

```sh
denon start
```
